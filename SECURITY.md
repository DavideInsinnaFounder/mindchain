# Security Policy

Mindchain is currently in an early-stage conceptual and non-production phase.

## Supported Versions
- The `/reference` implementation is **non-production** and **not security hardened**.
- No security guarantees are provided at this stage.

## Reporting a Vulnerability
If you believe you have found a security issue, please report it responsibly.

- Preferred: open a GitHub issue with minimal details and mark it clearly as **SECURITY**  
  (or use private channels if available in the future).

Please include:
- description of the issue
- steps to reproduce
- potential impact
- suggested mitigations (if any)

## Disclosure
We aim to acknowledge reports and coordinate a responsible disclosure timeline where applicable.
