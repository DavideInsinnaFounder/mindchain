# Glossary

**AI-to-AI**  
Transactions and value exchanges between autonomous AI agents.

**M2M (Machine-to-Machine)**  
Automated interactions between devices/systems without direct human involvement.

**Payment Rail**  
Infrastructure for transferring value (authorization, clearing, settlement), independent of a marketplace.

**Wallet (Agent Wallet)**  
A non-human wallet instance bound to an agent/device, enforcing spending rules.

**Service Credits**  
Programmable units used to pay for services (conceptually; not necessarily a speculative token).

**Intent**  
A structured request to transfer value, potentially signed and bound to constraints (expiry, nonce).

**Nonce**  
A unique value used to prevent replaying the same intent multiple times.

**Expiry**  
A time limit after which an intent becomes invalid.

**Ledger / Audit Log**  
A record of transactions used for transparency, monitoring, and dispute analysis.

**Settlement**  
Final reconciliation of balances (netting) on a chosen backend (on-chain/off-chain/fiat rails).

**Reference Skeleton**  
A minimal non-production codebase demonstrating feasibility without security guarantees.
